Environmental scan for manuscript project publication platform

Currently in the process of producing an environmental scan for a platform to publish the Archive of Early Middle English project, led by Dorothy Kim. The aim is to create a suitable platform where the AEME will be available for the public: this includes the TEI-encoded XML files, as well as digital images of the manuscripts, many of which are already available thanks to IIIF-integrated efforts by some libraries.

I'll write up some more thoughts on digital publication as the scan is more complete, but for now, here's a link to a quick test I built in Mirador 3, using some html and Glitch templates. (Thanks, Glitch!) The Mirador window should open up to four tiles presenting four AEME manuscripts.
